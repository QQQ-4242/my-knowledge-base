# Python & AI 常用英文词汇

| 英文 | 中文 | 用在哪 |
|------|------|--------|
| group | 分组 | df.groupby() |
| by | 按照 | group + by = 按……分组 |
| filter | 筛选 | 按条件过滤数据 |
| sum | 求和 | .sum() |
| category | 类别/分类 | 数据分类 |
| import | 导入 | import pandas |
| data frame | 数据框 | pd.DataFrame() |
| column | 列 | df["列名"] |
| row | 行 | 一行数据 |
| API | 应用程序接口 | 餐厅服务员模型 |
| request | 请求 | 你问服务器要东西 |
| response | 响应 | 服务器给你东西 |
