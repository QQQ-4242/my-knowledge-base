# 我的个人知识库

> 手机记 123 → Claude 周末整理 → 自动归档

## 目录结构（五层，不嵌套）

```
my-knowledge-base/
├── 0-memory/           ← Claude Code 自动记忆（勿手动改）
├── 1-经验库/            ← 调试教训、踩过的坑、解决方案
├── 2-灵感库/            ← 突然的想法、想做的东西、小说片段
├── 3-资源库/            ← 好文章链接、教程、工具推荐
├── 4-学习笔记/          ← 系统学习记录（AI、英语、造价、投资）
├── 5-项目复盘/          ← 每个项目做完后的总结
├── _index.md           ← AI 自动生成的总目录
└── README.md
```

## 手机记录格式（备忘录即可）

```
1 pip安装报错：UnicodeDecodeError，改utf-8解决
2 小说：反派动机——他以为主角杀了他的家人
3 https://xxx.com 一篇AI Agent教程，值得看
```

- `1 ` = 经验库
- `2 ` = 灵感库
- `3 ` = 资源库
- `4 ` = 学习笔记
- `5 ` = 项目复盘

## 整理规则

每周一次，把你的手机备忘录粘贴给 Claude，说"整理到知识库"。Claude 自动分类归档。
