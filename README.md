# 我的个人知识库

> 文件优先（File over App）— 全是 Markdown，不绑定任何工具
> AI 辅助维护 — Claude 帮忙分类、写摘要、更新索引

## 目录结构

```
my-knowledge-base/
├── memory/          ← Claude Code 记忆文件（自动加载）
├── 工程造价/         ← 专业课笔记、造价案例分析
├── AI学习/           ← AI 原理、开发经验、教程笔记
├── 小说素材/         ← 灵感、角色、情节片段
└── _index.md        ← AI 自动生成的总索引
```

## 使用方式

- Claude Code 的 memory 指向此仓库的 `memory/` 目录
- 所有内容以 Markdown 文件存储，用任何编辑器都能打开
- 推送到 GitHub 私有仓库，换设备 `git clone` 即用
