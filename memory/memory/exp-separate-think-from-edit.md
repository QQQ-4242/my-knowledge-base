# 表达与想法分离

**现象**：写东西时边想边改，永远出不来。改五版消息才发，精力耗在修正上而非原创想法上。

**根本原因**：把"创作模式"和"编辑模式"混在一起运行，两个任务互相打断。

**修复方法**：
1. 第一遍只管把想法倒出来，不纠错、不修饰
2. 倒完切到修改模式，最多改两遍
3. 两遍改完就发，不管满不满意

**Why:** 降低表达摩擦力，保护原创想法的产出量
**How to apply:** 每次写消息/文档/作业前，先决定"现在是创作模式还是修改模式"，不同时做
