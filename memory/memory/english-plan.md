# 英语学习计划

## 目标
- 日常英语：能用英文表达情绪、需求、生活场景
- 技术英语：能看懂 AI/编程文档，用英文搜技术问题

## 三条线

### 线 0：语法纠正（用户发英文时触发）
- 用户发英文 → 先纠正语法 → 再正常回复
- 格式：原文 → 修正版 → 简注

### 线 1：技术英语（对话中自动渗透）
- 解释技术词时拆词根（如 groupby = group + by）
- 中文变量名 → 提醒英文等价词
- 频率：每次对话自然出现

### 线 2：日常英语（双重触发）

**触发 A：对话开始时**
- 先复习上次的 3 个词（抽 1 个检验）
- 再给今天的 3 个新词，预告方向
- 例："上次的词：insomnia（失眠）。今天的：drowsy（昏昏欲睡的）/ unwind（放松）/ nightmare（噩梦）"

**触发 B：对话中自然插入**
- 聊到适合的场景 → 插入英文词 + 括号中文
- 频率：一个话题 1-2 个，轻话题才插，重话题（情绪崩溃、严肃讨论）不插
- 用户说"停" → 立刻收
- 不要求全记，记住多少算多少

**预期数量**：每次对话开始 3 新词 + 对话中自然出现 3-10 个。核心记住"开始 3 个"即可。

## 三阶段

### 阶段 1：词汇积累（现在~2周）
- 每轮对话积累 3 个日常词 + 若干技术词
- 目标：2 周后认识 20+ 日常词 + 20+ 技术词
- 检验：随机抽 5 个，能说出意思

### 阶段 2：句子表达（第 3-4 周）
- 每次给 1 个场景句子（真实生活，非课本）
- 目标：能说 10 句跟自己生活相关的完整句子

### 阶段 3：短对话（第 5 周+）
- 偶尔切 2-3 句英文再切回来
- 不追求语法，追求敢说
