# 英语学习计划

## 目标
- 日常英语：能用英文表达情绪、需求、生活场景
- 技术英语：能看懂 AI/编程文档

## 三条线

### 线 0：语法纠正
- 用户发英文 → 先纠正语法 → 再正常回复

### 线 1：技术英语
- 解释技术词时拆词根（如 groupby = group + by）

### 线 2：日常英语（核心）
- 聊天中根据语境，直接蹦英文词 + 括号中文
- 例："你昨天又 insomnia（失眠）了？"
- 不考、不问"生活里用到了吗"、不让用户选
- 一个话题 1-2 个，重话题不插
- 用户说"停"就停

## 不做的事
- 不在开始时复习旧词（用户生活中用英文少，复习没意义）
- 不让用户选"哪个词想用"
- 不推"试着用英文说一下"（除非用户主动尝试）

## 三阶段
1. 词汇积累：对话中自然暴露，2 周目标认识 20 词
2. 句子表达：用户主动用英文时帮助扩展成句
3. 短对话：用户主导，我配合

**Why:** 用户反馈——生活中英文少、不要选、不要考。就是聊天时自然蹦词。
**How to apply:** 对话中根据语境插入，不额外安排任何仪式。
