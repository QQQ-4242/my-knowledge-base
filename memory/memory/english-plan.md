# 英语学习计划

## 目标
- 日常英语：能用英文表达情绪、需求、生活场景
- 技术英语：能看懂 AI/编程文档，用英文搜技术问题

### 线 0：语法纠正（用户发英文时自动触发）
- 用户在对话中发了英文 → 我先纠正语法错误（如有），再回复正常内容
- 格式：原文 → 修改版 → 简单解释
- 不打断对话节奏，纠正完继续聊

## 双线执行

### 线 1：技术英语（对话中自动渗透）
- 我解释任何技术词时 → 必须拆词根给英文（如 group + by）
- 用户代码中出现了中文变量 → 我提醒英文等价词
- 频率：每次对话自然出现，不用额外安排
- 检验：下次对话我抽 3 个之前讲过的技术词

### 线 2：日常英语（每次对话结束时）
- 对话结束前三问 → Q2 "要不要科普" 同时附加：给 3 个日常高频词
- 选词标准：用户真实生活中会用的（情绪、睡眠、压力、日常对话）
- 不选课本词，选用户可能今晚就能用到的话
- 检验：下次对话开始或中间，抽 1 个上次的词

## 三阶段

### 阶段 1：词汇积累（现在~2周）
- 每轮对话积累 3 个日常词 + 若干技术词
- 目标：2 周后认识 20+ 日常词 + 20+ 技术词
- 检验：我能随机抽 5 个，用户能说出意思

### 阶段 2：句子表达（第 3-4 周）
- 每次给 1 个场景句子（真实生活，非课本）
- 例："I couldn't fall asleep last night."
- 目标：能说 10 句跟自己生活相关的完整句子
- 检验：给中文场景，说出英文句子

### 阶段 3：短对话（第 5 周+）
- 对话中偶尔切 2-3 句英文再切回来
- 不追求语法，追求"敢说"
- 目标：能用英文表达简单情绪和需求

**Why:** 用户需要日常英语不是技术英语，两条线同时走但不混淆。
**How to apply:** 我必须在每次对话中自动执行线 1（技术词），并在结束时执行线 2（3 个日常词）。
