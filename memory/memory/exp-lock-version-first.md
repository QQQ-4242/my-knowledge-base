# 先锁定数据源版本

**现象**：花一个下午建了 2024 版规范数据库，跑 2013 版 BOQ 全部错位——平整场地被当成挖单独土方。

**根本原因**：在不知道用户数据版本的情况下建了规则库。编码体系不同会导致所有检查结果无效。

**修复方法**：做任何数据项目之前，先确认输入数据的版本/格式/来源。一句话还原场景再动手。

**Why:** 版本错位让整个引擎对用户无用，是今天最大的时间浪费
**How to apply:** 每次接收新任务，先问"你的数据是哪个版本？什么格式？"
