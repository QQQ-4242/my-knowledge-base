# /analyze 命令

## 触发
- 用户输入 "/analyze" 或 "分析一下" 或 "分析本轮"
- 每轮对话结尾自动给浓缩版（3 bullet）

## 五维度

### 逻辑
- 怎么思考的，推理路径有什么变化

### 情绪
- 什么时候烦躁（频繁换词/感叹号/短句），什么时候流畅
- 识别什么场景消耗用户

### 决策速度
- 从"想做"到"动手"的间隔在缩短还是拉长

### 精力曲线
- 第几轮开始出错/沉默/敷衍，找到最佳收手时间

### 学习迁移
- 今天用的技能里哪些是近期刚学的
- 看到进步证据

## 输出格式

浓缩版（每次结尾自动）：
```
📊 今日：1个进步 | 1个卡点 | 1个建议
```

完整版（用户手动触发）：
```
📊 本轮侧写

逻辑：[推理路径变化]
情绪：[波动点+触发场景]
决策：[速度变化]
精力：[高峰期/低谷期]
迁移：[新技能应用证据]

🔧 行为调整
- [具体规则]
```

## 规则
- 时间用相对锚点（第X天/X号），不用"上次""前一阵"
- 没新洞察就跳过，不硬编
- 分析完必须有后续动作
- 浓缩版不占篇幅，不重复完整版内容
